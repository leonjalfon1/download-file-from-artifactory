# download-file-from-artifactory

### Usage

.\DownloadArtifact.ps1 -ArtifactoryServer "http://myserver:8081/artifactory" -ArtifactoryUser "myuser" -ArtifactoryPassword "mypassword" -ArtifactoryRepository "MyGenericRepo" -ArtifactPath "path/to/artifact" -DestinationPath "C:\path\to\save\filename.ext" -ArtifactVersion "latest"
